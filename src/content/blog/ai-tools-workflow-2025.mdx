工具本身不重要，工作流才重要。这篇文章分享的不是“最好的 AI 工具”，而是**我每天实际在用、产生了真实效果的配置**。

## 写作与内容

**Claude 3.7 Sonnet**（主力）

所有需要深度思考、长文写作、代码分析的任务，Claude 是我的首选。特别是它的 extended thinking 模式，处理复杂问题时比其他模型稳定很多。

**Perplexity Pro**（资讯研究）

当我需要了解一个新领域或者跟踪最新动态，Perplexity 比直接问 ChatGPT 可靠——因为它会给来源。

## 编程与工具开发

**Claude Code**（本地代码）

直接在终端调用，改代码效率极高。配合 Cursor 一起用，一个全局视野一个局部操作。

**Cursor**（日常开发）

AI 补全已经非常准了，特别是在你已有代码库的情况下。Composer 功能可以跨文件修改，大幅减少跳来跳去。

## 图像与设计

**Midjourney**（品牌图像）

还是最稳定的图像生成工具，特别是做有一致风格的系列图。

**Figma + 插件**（界面设计）

我不是设计师，但 Figma 的 AI 插件让我可以做出 80 分的界面，已经够用。

## 我的工作流串联方式

```
资讯收集 (Perplexity)
    ↓
思路整理 (Claude / 思维导图)
    ↓
内容创作 (Claude + 自己写)
    ↓
配图生成 (Midjourney)
    ↓
发布分发 (公众号 / 博客 / 小红书)
```

## 最被高估的工具

ChatGPT Plus。它依然很好，但按月订阅的性价比不如 API 调用或者 Claude Pro。如果你只能选一个，2025 年我会选 Claude Pro。

## 最被低估的工具

**Kollab**。作为 AI 助手的调度层，它把多个 Agent 的工作流串联起来，对于有多个并发项目的超级个体非常实用。这是我管理多个一人公司项目的核心工具。
